# quantum-plateau-bypass

量子AIのBarren Plateau（砂漠化）問題に対して、「戦わずに回避する」アプローチで取り組みました。量子回路出力を固定粒度で離散化する翻訳層（STE実装）を導入することで、量子-古典間のインターフェースで情報エントロピーを意図的に削減。過学習の抑制と勾配の安定伝播を同時に実現しています。4qubit/8qubit・最大4層の深い回路でも勾配消失は確認されず、Irisデータセットで正解率100%、Wineデータセット（13次元）で89%を達成しました。

## 1. はじめに
本アーキテクチャは、砂漠化問題の根本原因に対する直感的な仮説から出発し、AIとの対話的な検証を通じて設計されました。従来の量子ML研究とは異なる視点からアプローチしており、一部の説明は標準的な理論用語と異なる表現を用いている場合があります。

## 3. なぜこの設計なのか
砂漠化問題の根本原因が「**光量子の次元数があまりにも多く、頻度解析を行うだけのサンプルを集められない**」ことにあると考えました。
これは従来の「回数を増やす」「学習率を変更する」「学習データを増やす」では有限時間内での学習完了が実現不能になります。

そこで、思いついたのが「**WebP等の画像圧縮の考え方**」です。
人間の目では、PNGとWebPの違いをほぼ認識することは出来ませんがデータサイズは大幅に下げることが出来ます。
これと同じように、`1.000000` と `0.999999` にほぼ違いはありません。であれば、出来る限り近傍の数値は統合し、データの量を少なくすることで頻度を纏めさせることで学習を無理やり進めるというやり方が有効になるという考えです。

また、データ粒度について計算で出したり、動的に変化させない理由についてですが、学習データが動的に変化してしまうと学習が一向に進まなくなり有限時間内で終わらなくなります。
計算で出そうにも、古典AIが理解できる限界粒度、データから生み出される次元の組合せ、そのパターン数。これらはデータ、古典AIの性能によって変化し、次元の組合せとデータそのもので利用されてる数値との関連付けが不可能なため、粒度調査フェーズを設けることである程度自動的にフィッティングしてくれるようになっています。
（つまるところ現在はヒューリスティックなのでここには研究の余地があります）
